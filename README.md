

# Dayora

Dayora is an AI-powered, mood-based planner that helps users design their day, date, or trip in a simple and personalized way. It focuses on combining multiple planning needs into one platform, making it easier to decide what to do based on mood, preferences, and occasion.

## Overview

Planning daily activities, dates, or trips can be time-consuming and overwhelming. Dayora simplifies this by generating personalized plans based on how users want to spend their time. Whether someone is new to a city, planning a date, going out with friends, or organizing a trip, Dayora provides relevant suggestions and structured plans.

## Features

* Mood-based planning for personalized experiences
* Date planning with curated suggestions
* Trip planning with structured itineraries
* Recommendations for cafes, restaurants, and activities
* Playlist suggestions based on mood and context
* Budget-aware planning options

* HTML
* CSS

## Future Scope

Dayora can be extended with features such as AI-driven recommendations, real-time booking integrations, user accounts, saved plans, and social sharing capabilities.
